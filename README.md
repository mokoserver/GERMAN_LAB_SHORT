# GERMAN_LAB_SHORT

## Проект Олега Герман

Этот каталог представляет собой "сокращенную" версию проекта измерительной лаборатории, созданную для платформы MOKO SE по автоматизации измерений.

### Содержимое

*   **German's Measurement** и **German's Measurement Short**: Директории с результатами измерений (`.mpr`), сгруппированными по датам.
*   **data**: Содержит данные, включая `Results.csv` и схему измерительной установки.
*   **Python-скрипты**: Набор скриптов для автоматизации процесса измерений:
    *   `1 Настройка параметров.py`: Настройка параметров.
    *   `2 Инициализация приборов.py`: Инициализация оборудования.
    *   `3 Измерения.py`: Выполнение сокращенной последовательности измерений.
    *   `4 Создание отчета.py`: Генерация отчета.
*   **Библиотечные файлы**: Модули `MFRT.py`, `MGPH.py`, `MOKO.py`, `MOSC.py`.
*   **German's Measurement Short.msep**: Файл проекта MOKO SE для "сокращенной" версии.
*   **Отчет.docx**: Шаблон или пример отчета.

### Назначение

Проект `GERMAN_LAB_SHORT` является сокращенной версией измерительной лаборатории, предназначенной для проведения быстрых или укороченных циклов тестов на платформе MOKO SE.
